# Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo **MyPlaces - Weather** coleta, usa e protege as informações dos usuários.

Ao utilizar este aplicativo, você concorda com a coleta e o uso de informações de acordo com esta política.

---

## 1. Informações Coletadas

O aplicativo pode coletar os seguintes tipos de informações:

### 1.1 Informações de Localização
O aplicativo pode coletar dados de localização aproximada ou precisa do dispositivo do usuário para:

- Fornecer previsões do tempo corretas para a região atual
- Exibir alertas meteorológicos relevantes
- Melhorar a experiência do usuário

A localização é coletada **somente com o consentimento do usuário**, concedido por meio das permissões do sistema operacional.

### 1.2 Informações de Uso
Podemos coletar informações anônimas sobre o uso do aplicativo, como:

- Telas acessadas
- Interações com o app
- Dados de desempenho e falhas

Essas informações são utilizadas exclusivamente para melhorar a estabilidade e a funcionalidade do aplicativo.

### 1.3 Informações do Dispositivo
Podemos coletar informações técnicas do dispositivo, incluindo:

- Modelo do dispositivo
- Versão do sistema operacional
- Identificadores anônimos

Esses dados não identificam o usuário individualmente.

---

## 2. Uso das Informações

As informações coletadas são utilizadas para:

- Fornecer dados meteorológicos personalizados
- Melhorar a experiência do usuário
- Corrigir erros e melhorar a performance do aplicativo
- Cumprir obrigações legais e regulatórias

---

## 3. Compartilhamento de Dados

O aplicativo **não vende** dados pessoais dos usuários.

Os dados podem ser compartilhados apenas nos seguintes casos:

- Com provedores de serviços essenciais (como APIs de clima, analytics ou anúncios), quando aplicável
- Para cumprir exigências legais ou solicitações governamentais

Todos os terceiros seguem políticas de privacidade compatíveis com esta política.

---

## 4. Permissões do Aplicativo

O aplicativo pode solicitar as seguintes permissões:

- **Localização**: para fornecer previsões meteorológicas precisas
- **Internet**: para acessar dados meteorológicos atualizados
- **Notificações** (quando aplicável): para alertas de clima

O usuário pode revogar essas permissões a qualquer momento nas configurações do dispositivo.

---

## 5. Segurança dos Dados

Adotamos medidas técnicas e organizacionais razoáveis para proteger as informações contra acesso não autorizado, alteração, divulgação ou destruição.

---

## 6. Retenção e Exclusão de Dados

Os dados coletados são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política.

Caso deseje solicitar a exclusão de dados, o usuário pode entrar em contato pelo e-mail informado abaixo.

---

## 7. Privacidade de Crianças

Este aplicativo não é direcionado a menores de 13 anos. Não coletamos intencionalmente informações pessoais de crianças.

---

## 8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que os usuários revisem esta página regularmente para se manterem informados sobre eventuais mudanças.
